Existe correlaao espacial na qualidade dos miojos?

  • Para responder isso ser utilizado o ndice I de Moran, uma medida de correlao espacial que varia de -1 a 1.

  • Primeiro preciso criar a vizinhana de cada pas.

vizinhos <- 
  geo_ramen %>%
  group_by(Country) %>% 
  summarise(score = mean(Stars, na.rm = T)) %>% 
  poly2nb(.,
          queen=TRUE) #opcao para dizer que nao precisa "encostar"
  • Agora preciso criar pesos para cada vizinhana, no caso ser usado o mtodo mais simples que assume pesos iguais style = W.
vizinhos_peso <- nb2listw(vizinhos,
                          style="W",
                          zero.policy=TRUE) #permitir regies sem vizinhana
  • Teste de hiptese e coeficiente I de Moran.

Hiptese nula: No existe correlao espacial (I=0)

Hiptese alternativa: existe correlao espacial (I!=0)

geo_ramen %>%
  group_by(Country) %>% 
  summarise(score = mean(Stars, na.rm = T)) %>% 
  pull(score) %>% 
  moran.mc(., vizinhos_peso, zero.policy = T, nsim = 10000, alternative = "less")
## 
##  Monte-Carlo simulation of Moran I
## 
## data:  . 
## weights: vizinhos_peso  
## number of simulations + 1: 10001 
## 
## statistic = -0.018542, observed rank = 5077, p-value = 0.5076
## alternative hypothesis: less
  • Como o valor-p alto (0.5) e a estatstica prxima de 0, no h evidncias de correlao espacial em relao ao score. :(

Existe diferena significativa entre as qualidades dos miojos entre os continentes?

geo_ramen %>% 
  ggplot(aes(Stars)) +
  geom_histogram(bins = 15) +
  facet_wrap(~continent, scales = "free_y")

  • Como o comportamento da varivel score assimtrica e estranha primeiramente vou usar o teste no paramtrico de KruskalWallis para testar se ao menos 1 continente difere dos demais. As hipteses do teste so:

Hiptese nula: no existe efeito do continente no score

Hiptese alternativa: existe efeito do continente no score

kruskal.test(Stars ~ continent, data = geo_ramen)
## 
##  Kruskal-Wallis rank sum test
## 
## data:  Stars by continent
## Kruskal-Wallis chi-squared = 101.21, df = 5, p-value < 2.2e-16

  • Como o valor-p foi praticamente 0, rejeita-se a hiptese de que os scores no diferem de acordo com o continente.

  • Como sabemos que pelo menos algum continente difere dos demais, preciso testar par a par a fim de entender quem difere de quem. Para isso, usei o teste de Dunn para comparaes mltiplas.

  • Tentativa de grfico para comparaes mltiplas:

resultado_dunn <- 
  dunnTest(Stars ~ continent,
        data=geo_ramen,
        method="bh")

resultado_dunn$res %>% 
  mutate(p.value = round(P.adj,4)) %>% 
  select(Comparison, p.value) %>% 
  mutate(sig = p.value < .05,
         Comparison = as.character(Comparison),
         group = Comparison) %>% 
  separate(Comparison, into = c("C1", "C2"), sep = " - ") %>% 
  gather("ordem", "continent", -c(p.value,sig,group)) %>% 
  left_join(geo_ramen %>%
              group_by(continent) %>%
              summarise(score = mean(Stars, na.rm = T)) %>% 
              st_drop_geometry()) %>% 
  arrange(desc(score)) %>% 
  mutate(continent = factor(continent, unique(continent))) %>% 
  ggplot(aes(continent, score, group = group, col = sig)) +
  geom_line(aes(alpha = ifelse(sig, 1, .8)), show.legend = F) +
  geom_line(aes(x = 0, y = min(score))) +
  scale_color_manual("p.value<0.05",values = c("black", "firebrick2"))

  • A partir dos testes mltiplos foi possvel verificar que a qualidade dos miojos da sia superior aos da Amrica do Norte, Oceania e da Europa. Alm disso, tambm h superioridade dos miojos Norte Americanos quando comparados com a Europa.

  • No foi possvel afirmar nada em relao s comparaes da frica e Amrica do Sul, muito provavelmente pela baixa amostragem.
